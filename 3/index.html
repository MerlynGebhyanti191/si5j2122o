Pada pertemuan 3 saya menyimpulkan pelajaran antara lain:

  1. Membahas tentang materi yang berkaitan dengan UTS, bisa diakses dengan http://localhost/si5j2122o
  2. Jika koneksi gagal bukalah file wp-config.php klik kanan open with sublime/notepad, lihat ke baris 23 xampp control panel lalu klik admin. Jika sudah klik database dan cek db_si5j2122o buka isi database tersebut
  3. Membahas mengenai Posting, Jendela Posting ( Block Visual dan HTML ) tambah,ubah,hapus
  4. Klik pos tambah baru didalam dasbor lalu membuat judul bisa memanfaatkan bold,underline,link.
  5. Klik sunting sebagai HTML jika membutuhkan bloksetelah selesai sunting secara visual.
  6. Untuk menambahkan gambar buka digoogle lalu klik kanan pada gambar tersebut klik save image as, letakkan didalam folder cara manualnya klik unggah lalu terapkan gambar tersebut.
  7. Untuk mengupload video download aplikasi Everything. Download Instaler yang 64-bit. Kegunaan dari Everything itu untuk mencari file secara cepat lalu upload ke web server tersebut video tersebut bisa diambil didalam folder, dan di link youtube.
  8. Untuk cara mengubah pos tersebut, klik dasbor lalu klik pos kemudian sunting pos yang mau diubah lalu Untuk menghapus pos tersebut klik tong sampah.
  9. Status postingan (draft, publish, pending review, scheduled)
  10. Menambah postingan baru, membuat judul lalu klik simpan konsep, konsep inilah yang dinamakan dengan draft karena kita hanya punya ide judulnya saja Jika mau melanjutkan topik tersebut klik sunting.
  11. Publish dan menerbitkan pos jika sudah selesai
  12. Pending review (perlu ditinjau) peran pengguna misalnya bagian dari pelanggan, kontributor, penulis, dan editor. Jika ada pending review minimal diterbitkan oleh editor yang awalnya menulis sebagai kontributor jika sudah banyak editornya yang merevisi postingan tersebut .
  13. Scheduled mengatur agar terbit di tanggal lahir masing-masing.Dengan cara pos, tambah baru. Dengan contoh judul selamat ulang tahun, isi pos tersebut ucapan selamat ulang tahun. Jika sudah aturlah jadwal tanggal dan waktu terakhir klik jadwalkan.
  14. Kategori , tambahkan judul lalu klik pos, scroll ke bawah tambah kategori baru lalu terbitkan
  15. Tags/label seperti digunakan untuk pencarian
  16. - Komentar, sunting pos yang sudah di terbitkan lalu klik pos, jangan diberikan centang pada tulisan izinkan komentar
